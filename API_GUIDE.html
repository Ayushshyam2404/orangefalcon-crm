1. Overview

The External Data API exposes every collection in the Orange Falcon CRM database as read-only JSON endpoints. It also provides a real-time Server-Sent Events (SSE) stream that pushes live changes (inserts, updates, deletes) to any connected client the moment they happen.

This API is designed for external applications and dashboards that need to read CRM data without going through the internal user-login flow.

PropertyValue
Base URLhttps://74.208.5.193
ProtocolHTTPS (self-signed cert — see SSL note below)
AuthenticationAPI Key (header or query param)
Response formatJSON
Access levelRead-only for all collections
SSL Note: The server uses a self-signed certificate. If your HTTP client rejects it, disable certificate verification:
curl → -k  |  Node.js fetch/axios → rejectUnauthorized: false  |  Python requests → verify=False

GET /api/external/users

All CRM users. Sensitive fields (password, failed login attempts, lock info) are automatically stripped.

Query ParamDescription
—No filters available. Returns all users.

Key fields returned per user: _id, name, username, role (admin/staff), title, email, phone, online, clockedIn, lastLogin, sessionSeconds

4. Real-Time Stream (SSE)

GET /api/external/stream

Opens a persistent Server-Sent Events connection. The server pushes events in real time whenever any data in the CRM changes (new record, update, or delete).

Tip: On first connect, the server immediately sends a full snapshot event so your app has the current state before any change events arrive.

Event types

Event nameWhen firedPayload
snapshot Immediately on connect Full data dump (same as /snapshot endpoint)
change Any insert / update / delete in the DB { collection, operationType, documentId, document }
ping Every 30 seconds { ts } — keep-alive heartbeat

JavaScript / Browser example

// Works in any browser or Node.js (node-fetch or eventsource package)
const API_KEY = "ofcrm_56a712b8ecc434715ef8ba4020bae29518beacb98f9bdfdb8b4529a27d5b152c";
const es = new EventSource(
  `https://74.208.5.193/api/external/stream?apiKey=${API_KEY}`
);

// Initial full state
es.addEventListener("snapshot", (e) => {
  const data = JSON.parse(e.data);
  console.log("Full snapshot received:", data.users.length, "users");
});

// Live changes
es.addEventListener("change", (e) => {
  const { collection, operationType, documentId, document } = JSON.parse(e.data);
  console.log(`[${collection}] ${operationType} → id: ${documentId}`);
});

// Keep-alive
es.addEventListener("ping", (e) => {
  const { ts } = JSON.parse(e.data);
  console.log("ping", ts);
});

es.onerror = (err) => console.error("SSE error", err);

Python example

import sseclient, requests

API_KEY = "ofcrm_56a712b8ecc434715ef8ba4020bae29518beacb98f9bdfdb8b4529a27d5b152c"
url = f"https://74.208.5.193/api/external/stream?apiKey={API_KEY}"

response = requests.get(url, stream=True, verify=False)
client = sseclient.SSEClient(response)

for event in client.events():
    import json
    data = json.loads(event.data)
    print(event.event, data)

React / Next.js example

import { useEffect, useState } from "react";

const API_KEY = "ofcrm_56a712b8ecc434715ef8ba4020bae29518beacb98f9bdfdb8b4529a27d5b152c";

export default function Dashboard() {
  const [data, setData] = useState(null);

  useEffect(() => {
    const es = new EventSource(
      `https://74.208.5.193/api/external/stream?apiKey=${API_KEY}`
    );
    es.addEventListener("snapshot", (e) => setData(JSON.parse(e.data)));
    es.addEventListener("change",   (e) => {
      // Re-fetch just that collection or refetch full snapshot
      const { collection } = JSON.parse(e.data);
      console.log(collection, "changed — refetch if needed");
    });
    return () => es.close(); // cleanup on unmount
  }, []);

  return <pre>{JSON.stringify(data, null, 2)}</pre>;
}
